h1. Rounded Image

This library fixes the broken combination of images with the CSS3 border-radius style.

<img src="http://farm3.static.flickr.com/2712/4066296775_3ecbe37654.jpg" width="500" height="104" alt="rounded-image" />

h2. Usage

You can define CSS like so:

<pre class="code">
.myimagea {
  -moz-border-radius: 20px 0px 0px 20px;
  -webkit-border-top-left-radius: 20px;
  -webkit-border-bottom-left-radius: 20px;
  border-radius: 4px;
}
</pre>

You markup should look like this:

<pre class="code">
&lt;img class="rounded-image myimagea" width="160" height="120" src="form.jpg"&gt;&lt;/img&gt;
</pre>

When the page loads call the following:

<pre class="code">
RoundedImage.init();
</pre>
